# TWGS Collateral

This directory contains TradeWars 2002 Game Server validation material that is no longer
part of the primary SME review path for `space-wine`.

Contents:

- `docs/`: headless execution and profiling notes
- `tests/`: application-specific headless test container
- `results/`: captured TWGS before/after observations

The main repository surface is now centered on reproducible Wine behavior, standalone
tests, and versioned patch tracks for `wine-11.5` and `wine-10.0`.
